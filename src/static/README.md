# Локальные статические файлы интерфейса оператора

## html5-qrcode.min.js — библиотека сканирования QR камерой

Интерфейс сначала пытается загрузить `html5-qrcode.min.js` отсюда (локально,
для офлайн-цеха), и только при отсутствии — с CDN.

Для полностью автономной работы (цех без интернета) скачайте файл один раз
на компьютер с интернетом и положите сюда:

    curl -L -o html5-qrcode.min.js \
      https://cdn.jsdelivr.net/npm/html5-qrcode@2.3.8/html5-qrcode.min.js

Размер ~350 КБ. После этого сервер отдаёт библиотеку сам, интернет цеху
не нужен.

Если файла здесь нет — интерфейс возьмёт его с CDN (нужен интернет у сервера
или у телефонов при первой загрузке).
